This is our javaScript calculator for Project 5.

In order to preview the calculator, you can go to http://liveweave.com/ , copy and paste the javaScript (.js), CSS (.css),
and HTML (.html) files into their respective text areas (top right, bottom left, top left), and be sure that live mode 
(at the top of the page) is switched to "on".

This will allow you to preview and examine the calculator in the lower right hand side of the page.
